# 🎯 Étape 3 — Vos boucles (45 min)

**Le geste, toujours le même :** demandez un PLAN → validez-le → laissez produire → lancez `/verifier` → tranchez les écarts.

**À faire :**

1. Boucle 1 : le catalogue. Demandez le plan, corrigez-le si besoin, validez.
2. Production, puis `/verifier`. Lisez le rapport, vraiment.
3. Chaque écart : corriger maintenant, ou tracer pour plus tard. C'est VOUS qui décidez.
4. Boucle 2 si le temps : le détail + votre action.

**C'est fini quand :** votre catalogue s'affiche, conforme à VOTRE maquette, et vous avez tranché au moins un écart.

**Committez à chaque temps de boucle** : l'historique est votre filet.
**Bloqué ?** Levez la main, ou `git checkout step-3-boucles` pour repartir du fil rouge.
**Envie de creuser ce soir ?** `docs/NOTES-ETAPE.md` et les rapports `docs/boucles/`.
