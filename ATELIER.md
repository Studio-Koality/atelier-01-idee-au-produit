# 🎯 Étape 2 — Le visage de votre produit (20 min)

**À faire :**

1. Dérivez votre prompt de VOTRE spec — le canevas est dans `maquette/PROMPT-STITCH.md`.
2. Générez dans Google Stitch : 3 écrans (catalogue, détail + action, confirmation).
3. Itérez 2-3 fois max. Juste, pas joli.
4. Actez vos choix dans `maquette/REFERENCE-VISUELLE.md` : couleurs, règles, interdits.

**C'est fini quand :** les 3 écrans racontent le parcours de votre spec, et toujours zéro code.

**Stitch en panne ?** `maquette/maquette.html` fait foi (double-clic).
**Envie de creuser ce soir ?** `docs/NOTES-ETAPE.md`.
