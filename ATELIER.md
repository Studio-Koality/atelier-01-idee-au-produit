# 🎯 Étape 4 — En ligne (20 min)

**À faire :**

1. Poussez votre repo sur GitHub.
2. vercel.com → Add New → Project → votre repo → **Deploy**. Ne configurez rien.
3. Ouvrez votre URL depuis votre téléphone. Envoyez-la à quelqu'un.

**C'est fini quand :** votre produit vit sur une vraie URL.
